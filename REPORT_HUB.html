One-click usage

  • Double-click OPEN_REPORT_HUB.cmd to open this hub.
  • Double-click RUN_AND_OPEN_REPORTS.cmd to refresh data and then open reports.

If a rebuild fails

Close any open CSV or HTML preview tabs that may lock files in data/tables, then run RUN_AND_OPEN_REPORTS.cmd again.
